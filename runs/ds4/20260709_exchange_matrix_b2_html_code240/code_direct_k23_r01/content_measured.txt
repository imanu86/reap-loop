## Correctness Risks

1. **Race condition on expert loading**: `load_from_ssd(expert)` while another token may already be loading the same expert causes duplicate loads and potential cache corruption.
2. **Missing synchronization on cache check**: `cache.contains(expert)` and `load_from_ssd(expt)` are non atomic — a check-tup race allows double loads.
